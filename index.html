Basic Tools (and Tutorials)

  • Shell: bash, zsh. Top 10 shell commands, shellcheck, devhints.io/bash, Learn Bash in 10 minutes, The Unix Philosophy - Do One Thing and Do it Well. Zsh, ohmyz.sh
  • Text editor: vim (c.f. vimtutor). Neovim.io? VS Codium (built with Electron Framework)
  • VC: Git distributed version control system (VCS) for collaboration Intro to Github (1hr), Git Hello World, Git Cheat sheet, Connect with SSH, Public Key Cryptography
  • Webpage quickstart: Bootstrap HTML/CSS Graphic design patterns, HTML5, the Document Object Model (DOM), Cascading Style Sheets (CSS) => LESS, SCSS
  • Visualization and modeling: D3js.org uses SVG for rendering, — D3.js tutorials at ObservableHQ
  • Javascript/ECMAScript: MDN Reference Docs, Just enough to be dangerous with D3.js, Eloquent Javascript, Learning functional programming ES6 (1.5 hrs), MapReduce, D.Crockford: The wrrrld's most misunderstood programming language, How Javscript Works (Crockford, 2018), ECMAScript Standard and Brief Changelog, A Heretic's Perspective (1hr lecture) (Crockford, 2011)
  • Electron quickstart, Chrome extension quickstart (1-2hrs)
  • RESTful Web Application Architecture: web service API (Python Flask, Ruby Sinatra, or Node Express.js, for example), with a frontend framwork based on Model-View-Controller (MVC) design pattern and AJAX & JSON, such as React.js or Angular. ES6/Node/React/Webpack (5hrs tutorial), ESLint
  • Python3: IronPython shell. Scipy, Numpy. Flask (Simple REST web server). As opposed to RPC.
  • Learn to Program (U of Toronto, Coursera, ~25 hrs) Based on the Google Python course
